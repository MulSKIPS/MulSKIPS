This example illustrates the workflow to calibrate MulSKIPS CVD simulations considering gas-phase reactions  
- copy mulskips-source and pymulskips directory to this directory
- run the Jupyter notebook 

The notebook exploits the open-source software CANTERA gas-phase equilibrium solver.
Check the reference below for all details about CANTERA.
